About

Socka is a JavaScript dance and rythm game similar to Dance Dance Revolution. It supports up to two players.

It is compatible with the most common song formats (SM, DWI and SSC). The files can be retrieved from any compatible server over HTTP.

It also features support for gamepads and dancepads for more accurate gameplay.

You can find more technical information here.
